How It Works

Deputies uses composable tool calls. Here's a real example from this session:

// Delegate research to a subagent
subagent({
  agent: "explore",
  task: "Explore the repository structure..."
})

// Start a live preview server
service({
  action: "publish",
  port: 8080,
  label: "Web preview"
})

// Create a durable artifact
artifact({
  action: "create",
  path: "/workspace/report.md",
  type: "report"
})
